# architecture/
Implementation-independent architecture assets (KMOS-10020): diagrams, data
models, object/event catalogs, and reference architectures. The normative
architecture lives in `specifications/` and `reference/`; this directory holds
**derived diagrams only**.

**ADRs live in [`documentation/adr/`](../documentation/adr/) — that is the single,
canonical ADR home** (Coding Constitution §4). This directory intentionally holds
no ADRs; the empty `architecture/adr/` path is not used and is retained only to
avoid breaking historical references.
